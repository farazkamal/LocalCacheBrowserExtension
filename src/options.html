View Advanced Settings
Query strings to ignore, separated by new line (enter key)
You can ignore cache busting query strings, such as "_" used by jQuery, to increase the cache hit rate. This setting is case-sensitive.
Websites to ENABLE caching for, separated by new line (enter key)
You may only want to enable caching for certain websites. Default is to enable caching for all websites. Caching will only be enabled for websites with a URL that starts with a provided URL. This setting is NOT case-sensitive.
Ajax URLs to DISABLE caching for, separated by new line (enter key)
You may want to disable caching for certain ajax URLs. Default is to enable caching for all ajax URLs. Any ajax URL that starts with a provided URL will not be cached. This setting is NOT case-sensitive.
HTTP methods to DISABLE caching for, separated by new line (enter key)
You can disable caching for certain HTTP methods such as PUT and DELETE. This setting is NOT case-sensitive.
